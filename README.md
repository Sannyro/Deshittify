# Deshittify

Deshittify are css styles for Discord, reverting some of the changes from the visual refresh update.

## Themes
- Catppuccin Mocha
    - Modified <a href="https://github.com/catppuccin/discord">Catppuccin</a> mocha theme
- Ash
    - Default Ash theme
- Dark
    - Default Dark theme
- Onyx
    - Default Onyx theme
